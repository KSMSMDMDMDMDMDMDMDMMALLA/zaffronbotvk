# Zaffron

Игровой бот для сообщества ВКонтакте на Node.js и `vk-io`.

## Запуск бота

1. Установи Node.js 18.20 или новее.
2. Выполни `npm install`.
3. Создай `.env` по примеру `.env.example` и укажи `VK_TOKEN`.
4. Запусти `npm start`.

Главный файл бота: `src/index.js`.

## Mini App топ-10

Mini App находится в отдельной папке `miniapp-top`. Инструкция по сборке и
подключению виджета лежит в `miniapp-top/README.md`.

Папки `node_modules` не нужно загружать на хостинг: зависимости устанавливаются
командой `npm install`.
