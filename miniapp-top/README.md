# Zaffron Top Mini App

1. Создай Mini App в кабинете разработчика VK и скопируй ID приложения.
2. Скопируй `.env.example` в `.env` и укажи `VITE_VK_APP_ID`.
3. Укажи тот же ID в `vk-hosting-config.json`.
4. Выполни `npm install`, затем `npm run deploy` — сборка запустится автоматически.
5. Открой `https://vk.ru/app<ID приложения>` от имени администратора сообщества.
6. Нажми «Добавить приложение в сообщество» и выбери нужную группу.
7. Нажми «Получить токен виджета» и скопируй токен.
8. Нажми «Установить виджет в сообщество» (лучше открыть Mini App с компьютера).
9. На хостинге бота добавь `VK_APP_ID`, `VK_GROUP_ID` и `VK_WIDGET_TOKEN` в `.env`.
10. Перезапусти бота. Виджет обновится сразу и затем будет обновляться каждые 5 минут.

Не публикуй `VK_WIDGET_TOKEN` и не добавляй его в файлы Mini App.
